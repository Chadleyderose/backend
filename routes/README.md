# Routes

> The folder is linked with the controllers folder 
> **for more infor checkout controller README file**

``` 
Users Routes
  ├── GET all users
      ├── http://localhost:3000/users
  ├── GET user by { `id` }
      ├── http://localhost:3000/{ `id` }
  ├── PUT user by { `id` }
      ├── http://localhost:3000/{ `id` }
  ├── DELETE user by { `id` }
      ├── http://localhost:3000/{ `id` }

Blog Routes
  ├── GET all posts
      ├── http://localhost:3000/posts
  ├── GET post by { `id` }
      ├── http://localhost:3000/{ `id` }
  ├── PUT post by { `id` }
      ├── http://localhost:3000/{ `id` }
  ├── DELETE post by { `id` }
      ├── http://localhost:3000/{ `id` }

App Routes
  ├── POST register a user
      ├── http://localhost:3000/auth/register
  ├── POST Login user
      ├── http://localhost:3000/auth/login
  ├── GET verify
      ├── http://localhost:3000/auth/verify
  ├── Delete user
      ├── http://localhost:3000/users? { `id` }
  ├── Post post
      ├── http://localhost:3000/auth/posts
  ├── Post post
      ├── http://localhost:3000/auth/posts
  ├── Put post
      ├── http://localhost:3000/auth/posts/{ `id`}
      



```
